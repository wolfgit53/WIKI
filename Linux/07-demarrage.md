# Demarrage #

## Ordre de demarrage ##

![schema](img/ordredem.png)

### BIOS ###

**B**asic **I**nput **O**utput **S**ystem

Vérification:
* Processeur
* Carte mère

Vérification présence périphériques de boot:
* CD rom
* DVD rom
* Clef USB
* Disque dur

Active le MBR

### MBR - Boot primaire ###

**M**aster **B**oot **R**ecord

Placé sur le premier secteur adressable d'un disque dur:
* cylindre 0, tête 0 et secteur 1, ou secteur 0 en adressage logique
